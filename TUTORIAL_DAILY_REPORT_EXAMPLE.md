# Ondo Daily Report Generator —— 交互流程教程

> 以 2026-01-22 日报为例，展示用户与 AI 的完整交互流程。

---

## 概览

| 项目 | 内容 |
|------|------|
| 日报日期 | 2026-01-22（北京） / 2026-01-21（美东交易日） |
| 交互轮次 | 4 轮 |
| 最终产出 | `ondo_daily_report_2026-01-22.md` |

---

## 第一轮：启动

### 👤 用户输入
```
用模板帮我生成今天的 Ondo 日报
```

### 🤖 AI 响应

1. **确认日期**：北京时间 2026-01-22（周四）→ 美东 2026-01-21（周三）

2. **自动数据采集**（并行调用）：
   - Polygon API → SPY +1.15%, QQQ +1.12%, INTC +7.81%...
   - Ondo Tokens API → 108+ 支持标的
   - Web Search → 当日新闻热点

3. **生成日报初稿** → 保存为 `ondo_daily_report_2026-01-22.md`
   - 社区热议部分标记为"待补充"

4. **输出 Grok 提示词**：

```
搜索过去24小时 X/Twitter 关于以下话题的讨论（时间：2026-01-21 美东）：
1. Trump tariffs Greenland（特朗普关税政策转向+格陵兰协议）
2. Netflix earnings Q4 NFLX（Netflix财报表现）
3. Intel INTC earnings preview（Intel财报预览）
4. Stargate AI OpenAI SoftBank（Stargate AI项目）
5. Gold silver all time high（黄金白银新高）
6. 美股大盘走势（S&P 500, Nasdaq反弹）

每个话题返回：
- 热门标签（hashtags）
- 2-3条代表性推文（@用户名）
- 整体情绪（看多/看空/中性）
- 主要争议点
```

---

## 第二轮：提供 Grok 结果

> 📌 **用户操作**：复制上一步 AI 给出的 Grok 提示词，打开 [Grok](https://grok.x.ai) 进行查询，然后将返回结果粘贴回来。

### 👤 用户输入
```
grok的[1. Trump tariffs Greenland
热门标签：#Trump、#Greenland、#Tariffs、#NATO
代表性推文:
@xray_media "特朗普的'协议'是关税勒索..."
@Louis74330106 "一切都与矿产资源权有关..."
整体情绪: 看空
主要争议点: 胁迫 vs 外交；NATO盟友关系...

2. Netflix Q4 NFLX
热门标签：#NFLX、#Earnings、#Streaming
...（其余4个话题省略）
```

### 🤖 AI 响应

1. **整合到"社区热议"章节**：
```markdown
## 2.5 社区热议 (X/Twitter)
### 🔥 热度最高：特朗普关税政策转向与格陵兰协议
| 代表性观点 | 来源 | 情绪 |
|-----------|------|------|
| "一切都与矿产资源权有关..." | @Louis74330106 | 🔴 |
**整体情绪**：🔴 看空 — 大多数帖子批评...
```

2. **输出 OpenAI 提示词**：

```
搜索过去24小时美股热点新闻（2026-01-21），按以下分类整理：

1) 宏观政策
- 特朗普达沃斯论坛演讲要点
- 格陵兰协议框架细节
- 关税政策最新动态
需要：来源+UTC时间、政策要点、市场影响、风险点

2) 科技股动态
- Stargate AI项目最新进展
- Intel Q4财报预期与看点
需要：来源+时间、影响链条、风险点

3) 重要财报
- Netflix Q4详解（订阅数、广告收入、2026指引）
需要：核心KPI、市场反应、风险点

4) 大宗商品
- 黄金突破$4800驱动因素
- 白银涨幅超黄金原因
需要：价格水平、驱动拆解、风险点

优先引用 Reuters 带时间戳的报道。
```

---

## 第三轮：提供 OpenAI 结果

> 📌 **用户操作**：复制上一步 AI 给出的 OpenAI 提示词，打开 [ChatGPT](https://chat.openai.com) 进行查询，然后将返回结果粘贴回来。

### 👤 用户输入
```
以下是openai的

## 1) 宏观政策
### 1.1 特朗普达沃斯论坛演讲要点
来源：Reuters 2026-01-21 14:21 UTC
要点：对格陵兰要求"立即谈判"，强调"不会使用武力"...

### 1.2 格陵兰"协议框架"细节
来源：Reuters 2026-01-21 19:43 UTC
内容：与NATO达成"格陵兰及整个北极地区"框架，具体条款未披露...

### 1.3 关税政策
- 格陵兰相关：原定2/1关税不实施
- 瑞士关税：从39%降至15%...

## 2) 科技股动态
...（Intel数据中心预期+30%、18A良率风险等）

## 3) 重要财报
...（Netflix暂停回购、$590亿过桥贷款等）

## 4) 大宗商品
...（黄金盘中$4,887.82、白银高beta特征等）
```

### 🤖 AI 响应

1. **整合深度分析到各章节**：
   - 热点1：新增瑞士关税细节、风险提示
   - 热点2：新增Netflix并购战、暂停回购信息
   - 热点3：新增Intel数据中心+30%预期、18A风险
   - 宏观环境：新增黄金驱动因素拆解

2. **更新参考链接**：新增 Reuters 带时间戳链接

3. **执行最终验证**：
```
✅ 无 {{ 占位符
✅ 无 }} 占位符
✅ 无 '待填' 占位符
✅ 无 '示例' 占位符
✅ 无 '待补充' 占位符
```

4. **输出完成确认**：
```
✅ 日报最终版完成！
文件路径: ondo_daily_report_2026-01-22.md
总行数: 412 行
```

```
提示词优化下，grok和openai的反馈分两次给，避免上下文太大
```

### 🤖 AI 响应

更新 `ONDO_DAILY_REPORT_TEMPLATE_PROMPT.txt`：

| 原流程 | 优化后 |
|--------|--------|
| 第四步：同时给 Grok + OpenAI 提示词 | 第四步：只给 Grok 提示词 |
| 第五步：用户一次性返回两个结果 | 第五步：Grok 结果后再给 OpenAI 提示词 |

```

        ↓

        ↓
   AI: [整合深度分析] → [最终验证] → [完成]
        ↓
用户: "优化提示词流程"
        ↓
   AI: [更新模板文件]
```
